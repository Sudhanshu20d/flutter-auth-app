# 🔐 Flutter Auth App

A simple Flutter application demonstrating **Login & Signup functionality** using a dummy API.

---

## 🚀 Features

* Login Screen
* Signup Screen
* Form Validation
* API Integration (Reqres API)
* Loading Indicator
* Error Handling
* Token Storage using SharedPreferences
* Navigation to Dashboard after Login

---

## 🛠️ Tech Stack

* Flutter
* Dart
* HTTP package
* Shared Preferences

---

## 📡 API Used

* https://reqres.in
